# 嵌套

语法：

```scss
main {
    background-color: #000;

    a {
        text-decoration: none;

        &:hover {
            text-decoration: underline;
        }
    }
}
```

编译后：

```css
main {
  background-color: #000;
}

main a {
  text-decoration: none;
}

main a:hover {
  text-decoration: underline;
}
```
